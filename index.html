407 CIP Leaning Pole or Potential Overload

Reference image unavailable

When to use Code 407

  • J suffix pole in Geocall
  • Leans 10°+ from vertical (7' off center for 45' pole)
  • Requires guying, anchoring, or reinforcement
  • Shows bowing or distortion

CIP-Only Poles

When CIP Poles Require Code 407

Use code 407 for CIP-only poles within 3 spans when they adversely affect an SDG&E pole.

Required Documentation

  • CIP pole name and location
  • Detailed description of how it affects SDG&E pole
  • Span distance and condition notes

Identification Rule

Only CIP poles up to 3 spans away can be reported with code 407. Measure the span distance carefully and document the connection.

Quick recap

Code 407 Summary

✓ Use on J suffix poles that lean more than 10° from vertical.

✓ Use when poles require guying, anchoring, or reinforcement, or show bowing and distortion.

✓ Use for CIP-only poles within 3 spans that adversely affect SDG&E poles.

→ Always include CIP name and description in notes.
